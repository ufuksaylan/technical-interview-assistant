# Technical Interview Assistant

## Setup

```sh
npm install
```

Create `.env.local`:

```sh
VITE_RAPIDAPI_HOST=
VITE_RAPIDAPI_KEY=
VITE_OPENAI_API_KEY=
VITE_OPENAI_BASE_URL=https://api.openai.com/v1
VITE_OPENAI_MODEL=gpt-3.5-turbo
```

## Commands

```sh
npm run dev
npm run build
npm run test:unit
```
